  • teraPad
    • TeraPad(テラパッド)は、Windowsで動作するSDIタイプのテキストエディター。名前の由来は作成した寺尾 進さん。20年以上前から存在しているソフトウェア。軽量な動作が特徴的な、シンプルな作り。寺尾 進と聞くと俳優の寺島 進さんと一瞬間違える。
  • Visual Studio code
    • マイクロソフトによって2015年にリリースされているソースコードエディタ。Windowsのほかに、Mac OS、Linux上でも動作。シンタックスハイライト、インテリセンス（自動補完システム）やデバッグ機能もある。
  • XREA
    • XREA（エクスリア）は eXtreme Realm for Experts and All の略。GMOデジロックが運営。ユーザー側で自由に設定をカスタマイズできる、高機能なレンタルサーバー。最新のCPUとSSD搭載で高速化を実現。また、安定したサイト運営が可能。全プランで無料独自SSLに対応。ただ、自由度が高い分、初心者向けとは言えないかもしれない。
    　XREA Free　→　広告表示の無料プラン
    　XREA Plus　→　広告非表示の有料プラン（月額191円～）
  • FFFTP
    • Webサイトを公開するときに使うFTPフリーソフト。Windows環境の定番ツール。FTPはFile Transfer Protocolの略。サーバーとクライアント間で、ファイルを送受信する通信の決まり事（ルール）の意味。サーバーにアップされたファイル一覧を確認したり、ファイルをサーバーへアップロード、ダウンロードもできる。
  • paint.net
    • 画像編集フリーソフト。Windows用にワシントン州立大学のプロジェクトとして始まり、マイクロソフトの指導と共に開発。シンプルで直感的なユーザーインターフェースで初心者でも分かりやすい。
